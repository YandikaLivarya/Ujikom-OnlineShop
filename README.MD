# S*CKSOCKS - Fullstack Sock Brand Website 🧦

Proyek website e-commerce brand kaos kaki "S*CKSOCKS" yang dibangun untuk kebutuhan Ujikom. Website ini memiliki fitur manajemen inventory, checkout real-time, dan integrasi payment gateway Xendit.

## 🚀 Fitur Utama
- **Frontend**: React.js, Tailwind CSS (Skena Aesthetic).
- **Backend**: Node.js, Express.js, MongoDB.
- **Payment**: Integrasi Xendit (Test Mode).
- **Real-time**: Auto-refresh status pesanan via Ngrok Webhook.
